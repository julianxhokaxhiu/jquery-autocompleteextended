AutoComplete Extended

A simple plugin for having multiple autocomplete inputs and more

Credits

  • Julian Xhokaxhiu, developer
  • Aicod srl for the time they let me to develop this :)

Features

  • It simply works with jQuery and jQueryUI ( + theme), nothing else is required.
  • It expands the already fully working AutoComplete widget of jQueryUI, but adding more options.
  • It preservers the "name" attr of the input, if you use it in multiple mode (useful for example, if you need to post some tags all on one submit).
  • Much more (feel free to discover them by yourself) :)

Documentation

This option define where the autocomplete should point to, when remoting getting the items. If it's relative, it points to the root path, but it can also be absolute.
  • String
  • '' (Empty String)
  • $('input').autocompleteextended({SearchUri:'search.php'});
This option define how many characters must be in the input before the search in the remote source begins.
  • Integer
  • 2
  • $('input').autocompleteextended({minLength:4});
This option define if you want to use regex patterns on the text you insert (only on the input, NOT on the new windows and NOT on the list window), for example if you want only to have numbers.
  • Boolean
  • false
  • $('input').autocompleteextended({useRegEx:true});
This option define the pattern to use with the regex. IT MUST BE USED IN CONJUCTION WITH useRegEx TO MAKE IT WORK.
  • RegExp Object
  • null
  • $('input').autocompleteextended({RegExPattern:/^([0-9]+)$/});
This option define if the input must act as a multiple one. See the examples to make it more clear.
  • Boolean
  • false
  • $('input').autocompleteextended({multiple:true});
This option define the default value(s) for the input at the initialization time. If you're using it in conjuction with the multiple option, you must pass it as an Array.
  • String or Array
  • null
  • $('input').autocompleteextended({multiple:false,defaultvalue:'Default value'}); or $('input').autocompleteextended({multiple:true,defaultvalue:['Default','value']});
This option define if you want the List button next your input (this allows to scroll inside ALL the elements that you're filtering into).
  • Boolean
  • true
  • $('input').autocompleteextended({withList:false});
This option define if you want the New button next to your input (this allows to insert values that are NOT into the remote list).
  • Boolean
  • true
  • $('input').autocompleteextended({withNew:false});
This option define a function callback that will be executed when ANY window will be closed.
  • function
  • null
  • $('input').autocompleteextended({callback:function(){alert('Callback!')});
This group define the options regarding the buttons next to the input (like the New, List or Delete button.
  • This option define if you want to have icons on the buttons or if you want text (not both of them).
    • Boolean
    • true
    • $('input').autocompleteextended({buttons:{useIcons:false}});
  • THis option define the text of the New button.
    • String
    • 'New'
    • $('input').autocompleteextended({buttons:{textNew:'New item'}});
  • This option define the text of the List button.
    • String
    • 'List'
    • $('input').autocompleteextended({buttons:{textList:'List of items'}});
  • This option define the text of the Delete button.
    • String
    • 'Delete'
    • $('input').autocompleteextended({buttons:{textDelete:'Delete item'}});
This group define the options regarding certains actions on the relative windows.
  • This group define the callback options for the add window.
    • This option define the callback when OK is pressed.
      • function
      • null
      • $('input').autocompleteextended({windows:{add:{onOK:function(){alert('OK Callback!')}}}});
    • This option define the callback when Cancel is pressed.
      • function
      • null
      • $('input').autocompleteextended({windows:{add:{onCancel:function(){alert('Cancel Callback!')}}}});
    • This option define the callback when the window is closed.
      • function
      • null
      • $('input').autocompleteextended({windows:{add:{onClose:function(){alert('Cancel Callback!')}}}});
  • This group define the callback options for the list window.
    • This option define the callback when OK is pressed.
